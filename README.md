# AfriFuta Project Setup Instructions

Follow these steps to set up and run the project locally.

```bash
# 1. Clone the Repository
# Clone the project repository from GitHub and enter its directory.
git clone https://github.com/YOUR_USERNAME/afrifuta.git
cd afrifuta

# 2. Install Dependencies
# Install all the project dependencies using your package manager.
# Choose either npm or bun depending on what you have installed.

npm install
# or
bun install

# 3. Run the Development Server
# Start the development server with hot reloading.
# The app will be accessible at http://localhost:5173 in your browser.

npm run dev
# or
bun run dev

# 4. Build for Production (Optional)
# Generate optimized production-ready files in the 'dist' folder.

npm run build
# or
bun run build
```
